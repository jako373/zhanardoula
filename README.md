# Zhanar Doula Atyrau

Жанардың Атырау қаласындағы кәсіби доула қызметіне арналған жеке Next.js сайты.

## Іске қосу

1. `pnpm install` орындаңыз.
2. `pnpm dev` арқылы жергілікті серверді іске қосыңыз.

## Командалар

- `pnpm dev` — жергілікті әзірлеу сервері
- `pnpm test` — негізгі мазмұн тексерістері
- `pnpm lint` — ESLint тексерісі
- `pnpm build` — Vercel-ге арналған production құрастыру

## Қызметтер

- Next.js 16
- WhatsApp арқылы өтінім жіберу
- Vercel hosting

Сайт дерекқорсыз жұмыс істейді және келушілердің өтінімдерін серверде сақтамайды.
